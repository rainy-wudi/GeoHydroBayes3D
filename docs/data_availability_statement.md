# Draft data availability statement

The public repository includes a fully synthetic demonstration dataset that allows reproduction of the complete computational workflow, including sample construction, KD-tree mapping, PA-DNN surrogate training, model comparison, global sensitivity analysis, and Bayesian inversion. The dataset preserves the parameter ranges, data schema, borehole count, and hydraulic-head response structure used in the manuscript.

The full raw engineering investigation data, original topographic files, and proprietary finite-element output files are not publicly available because of project confidentiality restrictions.
