# Abaqus template input files

Place the Abaqus template input file used for random-field assignment in this directory or in the working directory from which the MATLAB scripts are executed.

The original local template was named `Job-0.inp`. Large Abaqus `.inp`, `.inc`, `.odb`, `.dat`, `.msg`, `.sta`, `.sim`, and generated random-field output files are intentionally not included in this public source-code package because they are project-specific generated files and can be very large.
